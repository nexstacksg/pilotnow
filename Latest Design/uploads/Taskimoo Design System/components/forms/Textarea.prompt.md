Multiline text input — matches Input's chrome (mono label, hint/error, red focus ring) with a vertically resizable field.

```jsx
<Textarea label="Description" rows={5} placeholder="What needs to ship…" />
<Textarea label="Acceptance criteria" invalid hint="Required before moving to QA" />
```

Notes: only vertical resize is enabled. Use for descriptions, acceptance criteria, and notes — keep labels in the mono uppercase voice.
